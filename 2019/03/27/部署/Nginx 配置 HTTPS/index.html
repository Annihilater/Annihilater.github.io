Ubuntu 中使用 Nginx 配置 HTTPS

获取证书

可以从域名服务商处获得免费的域名证书，自己的域名服务商是腾讯云，在腾讯云获得的免费证书如下形式：

Nginx 文件夹内获得 SSL 证书文件 1_www.domain.com_bundle.crt 和私钥文件 2_www.domain.com.key。
1_www.domain.com_bundle.crt 文件包括两段证书代码 “—–BEGIN CERTIFICATE—–” 和 “—–END CERTIFICATE—–”,
2_www.domain.com.key 文件包括一段私钥代码 “—–BEGIN RSA PRIVATE KEY—–” 和 “—–END RSA PRIVATE KEY—–”。

证书安装

证书上传到服务器

上传证书可以使用 scp 命令：

使用 scp 命令无法传到远程服务器的权限目录下，可以先将文件传到不需要读写权限的目录，再使用 mv 命令将其移动到 nginx 目录（这个是需要 sudo 权限的目录）。当然存放证书的目录并不是规定的，你可以随意选择放在哪里，只要在 nginx 配置中可以读取到它们就可以了。

将域名 www.domain.com 的证书文件 1_www.domain.com_bundle.crt 、私钥文件 2_www.domain.com.key 保存到同一个目录，例如 /etc/nginx/crt 目录下。
