一、Python 是如何进行内存管理的？

答：从三个方面来说，对象的引用计数机制、垃圾回收机制、内存池机制

  1. 对象的引用计数机制
    Python 内部使用引用计数，来保持追踪内存中的对象，所有对象都有引用计数。
    引用计数增加的情况：

    1. 一个对象分配一个新名称；
    2. 将其放入一个容器中（列表、元组或字典）

    引用计数减少的情况：

    1. 使用 del语句对对象别名显示的销毁
    2. 引用超出作用域或被重新赋值

    sys.getrefcount()函数可以获得当前对象的当前引用计数多数情况下，引用计数比你猜测的要大得多。对于不可变数据（数字、字符串、元组），解释器会在程序的不同部分共享内存，一遍节约内存。

  2. 垃圾回收

    1. 当一个对象的引用计数归零时，它将被垃圾收集机制处理掉；
    2. 当两个对象 a 和 b 相互引用时，del 语句可以减少 a 和 b 的引用计数，并销毁用于引用底层对象的名称。

    然而由于每个对象都包含一个对其他对象的应用，因此引用计数不会归零，对象也不会销毁。从而导致内存泄漏，为解决这一问题，解释器会定期执行一个循环检测器，搜索不可访问对象的循环并删除他们。

  3. 内存池机制
    Python 提供了对内存的垃圾收集机制，但是它将不用的内存放到内存池而不是返回给操作系统。

    1. Pymalloc机制：为了加速 Python 的执行效率，Python 引入了一个内存池机制，用于管理对小块内存的申请和释放；
    2. Python 中所有小于256字节的对象都使用Pymalloc实现的分配器，而大的对象则使用系统的 malloc；
    3. 对于 Python 对象，如整数、浮点数和 List，都有其独立的私有内存池，对象间不共享他们的内存池。也就是说如果你分配又释放了大量的整数，用于缓存这些整数的内存就不能再分配给浮点数了。

二、什么是 lambda 函数？它的好处是什么？

答：lambda 表达式，通常是在需要一个函数，但是又不想费神去命名一个函数的场合下使用，也就是指匿名函数。

优点：是一个短小的回调函数

六、Python 如何拷贝一个对象（赋值、浅拷贝、深拷贝的区别）

答：

  1. 赋值：就是创建了对象的新引用，修改其中任意一个变量都会影响到另一个对象。
  2. 浅拷贝：创建一个新的对象，但它包含的是对原始对象中包含项的引用
    （如果用引用的方式修改其中一个对象，另一个对象也会改变）
    浅拷贝的方式：
    • 完全切片方法
    • 工厂函数：list()等
    • copy 模块的 copy()函数
  3. 深拷贝：创建一个新对象，并且递归地复制它所包含的对象
    （修改其中一个，另一个不会改变）
    深拷贝的方式：
    • copy模块的deepcopy()函数

七、介绍一下 except 的用法和作用

执行 main_action，如果引发异常，则执行过程会跳到 except 语句。对每个 except 分支顺序尝试执行，如果引发的异常与 except 中的异常匹配，则执行相应的语句；

如果所有 except 都不匹配，则异常会传递到下一个调用本代码的最高层 try 代码中。

如果没有执行 main_action 没有引发异常，则执行 else_block；

最后无论之前执行了什么、有没有引发异常、异常有没有处理，都会执行 finally_block。

finally 并不终止异常，如果异常没有被处理，还会继续往后传递。

答：pass 不会执行任何操作，一般作为占位符或者创建占位程序

十、如何使用 Python 查询和替换一个文本字符串？

答：

  1. 可以使用 re模块中的sub()函数或者subn()函数来进行查询和替换。

    参数：

    • pattern : 正则中的模式字符串。
    • repl : 替换的字符串，也可为一个函数。
    • string : 要被查找替换的原始字符串。
    • count : 模式匹配后替换的最大次数，默认 0 表示替换所有的匹配。

    subn()方法执行的效果跟sub()一样，不过它会返回一个二维数组，包括替换后的新的字符串和总共替换的数量。

  2. 字符串内置replace()方法

十一、Python 里面match()和search()的区别？

re.match 尝试从字符串的起始位置匹配一个模式，如果不是起始位置匹配成功的话，match()就返回none。

re.search 扫描整个字符串并返回第一个成功的匹配。

十二、用 Python 匹配 HTML tag 的时候，<.*> 和 <.*?> 有什么区别？

答：贪婪模式和非贪婪模式的区别。不加 ? 的为默认贪婪模式，加 ？的为非贪婪模式。

十三、Python 里面如何生成随机数？

答：random模块

随机整数：

  • x = random.randint(a, b)：a < x < b
  • x = random.randrange(a, b, c)
    • a：start
    • b：stop
    • c：step

随机实数：

  • x = random.random()：x 为随机实数
  • x = random.uniform(a, b)：a < x < b
    • a：start
    • b：stop

十四、有没有一个工具可以帮忙查找 python 的 bug 和进行静态代码分析？

答：PyChecker 是一个 python 代码的静态分析工具，它可以帮助查找 python 代码的 bug，会对代码的复杂度和格式提出警告。

Pylint 是另一个工具可以进行 codingstandard 检查。
