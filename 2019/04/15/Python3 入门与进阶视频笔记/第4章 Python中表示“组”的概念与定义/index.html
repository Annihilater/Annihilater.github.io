第4章 Python中表示“组”的概念与定义

python
入门与进阶
发布日期:   2019-04-15
更新日期:   2019-12-06

本章详细讲解了“组”的概念，以及在Python中用来表示“组”的一些类型，包括：元组、列表、集合和字典。

4-1 列表的定义

列表的定义是使用[]，中间元素使用，隔开

嵌套列表：

在其他语言里叫做二维数组

4-2 列表的基本操作

切片操作：

  • 当切片操作取出的是一个元素时：返回的类型为该元素的类型，元素类型可能是字符串、数字、字典、列表、元组、集合等等
  • 当分片操作取出的是多个元素时：返回的是列表类型

这就造成了返回类型不可控的局面，慎重考虑！

列表有加法和乘法，没有减法：

4-3 元组

元组的定义是使用()，中间元素用,隔开

元组支持切片操作

元组有加法和乘法，没有减法

有趣的现象：元组只有一个元素的时候

因为()即表示元组，同样也是数学运算的括号，

Python硬性规定：如果括号内只有一个元素，则将其识别为数学运算的括号。

那么问题来了，如何定义只有一个元素的元组呢？

4-4 序列总结

str字符串、list列表、tuple元组都是序列

序列里的每个元素都会分配一个序号，序列支持切片操作。

序列支持加法、乘法，不支持减法、除法。

序列还支持 in、not in操作：

len、max、min 操作

如何将字符转换成ASCII 码？使用 ord()函数

4-5 set 集合

set的定义

  • 使用{}，中间元素使用，隔开

set的特点

  • 特点一：无序

  • 特点二：不重复

set的普通操作

  • len、in、not in

set的特色操作

  • 差集：-

  • 交集：&

  • 合集或者并集：|

定义空set

set访问内部元素

set中的元素获取是通过迭代实现的，具体实现课采取两种方法：

  1. 通过iterator()
  2. 泛型中的迭代处理

4-6 dict 字典

dict的定义

  • 使使用{}，中间的元素时key:value形式，元素之间使用，隔开

dict的类型

  • 字典是集合类型，是无序的。

dict最常用的操作

  • 通过 key来访问 value

dict的重要特性

  • dict是不能有相同key，后加进来的相同key的value会将之前的 value覆盖；

  • dict的 key是分类型的；

  • dict的 value可以使任意类型：list、number、tuple、str、dict、set等等；

  • dict的 key必须是不可变的类型：number、str、tuple

Python 中的数据类型按照可变与不可变分类：

  • 可变数据类型：list、dict、set
  • 不可变数据类型：number、str、tuple
