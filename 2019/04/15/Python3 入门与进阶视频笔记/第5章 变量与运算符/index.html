本章详细讲解变量的意义与七种运算符，并对每一种运算符的扩展做出详细的讲解。

5-1 什么是变量

变量命名的可读性要强！变量在 Python里就是一个名字。

Python的变量是不需要定义的。

5-2 变量的命名规则

  • 变量命名只能使用数字、字母、下划线；
  • 首字符不能为数字；
  • 系统关键字不能用在变量名中；（系统关键字被称为保留关键字）
  • 变量名是区分大小写的；
  • Python的变量是没有类型限制的，任何类型都可以赋值给变量，是 Python 作为动态语言的特性之一。

类型区别

不可改变类型（值类型）在需要更改变量内容的时候会抛弃原对象，指向新的对象；

可变类型（引用类型）在更改变量内容的时候则是直接在原对象上修改。

5-4 列表的可变与元组的不可变

小知识：

  • id()函数可以获取对象的内存地址；
  • 计算机内存地址是用十六进制来表示的。

列表的可变

如以上过程所演示，列表在改变内部元素的时候，内存地址是没有改变的，也就是说变量名指向的是同一个对象，这就是列表的可变性。

元组的不可变

上述过程表示元组的不可变。如果需要改变变量名必先抛弃原对象，重新指向新对象。

不可变类型的重要意义

编码最重要的是要保证代码的稳定性，简单代码变量的可变不可变意义不大，因为都是自己随手写的，别人不可能来调用或者来修改代码的，但是问题是在一个复杂的项目里，我们不可能一个人把一个很复杂的项目都写完，必须是很多人共同来写代码。如果是共同来写代码的话就涉及到代码的调用，涉及到代码的调用，这些不可能改变的类型是有优势的，因为别人尝试修改不可改变的类型马上就会报错，根本不会让它运行，这就避免了一些隐藏的错误。

建议：能用不可改变的类型就用不可改变的类型。当然如果你需要动态改变组内部元素的话，还是用可变类型比较合适。

  • /

  • //

  • **

5-6 赋值运算符

运算符 描述 实例
= 简单的赋值运算符 c = a + b 将 a + b 的运算结果赋值为 c
+= 加法赋值运算符 c += a 等效于 c = c + a
-= 减法赋值运算符 c -= a 等效于 c = c - a
*= 乘法赋值运算符 c *= a 等效于 c = c * a
/= 除法赋值运算符 c /= a 等效于 c = c / a
%= 取模赋值运算符 c %= a 等效于 c = c % a
**= 幂赋值运算符 c **= a 等效于 c = c ** a
//= 取整除赋值运算符 c //= a 等效于 c = c // a

5-7 比较运算符

不同类型的对象不能进行比较！

运算符 描述 实例
== 等于 - 比较对象是否相等 (a == b) 返回 False。
!= 不等于 - 比较两个对象是否不相等 (a != b) 返回 true.
<> 不等于 - 比较两个对象是否不相等 (a <> b) 返回 true。这个运算符类似 != 。
> 大于 - 返回x是否大于y (a > b) 返回 False。
< 小于 - 返回x是否小于y。所有比较运算符返回1表示真，返回0表示假。这分别与特殊的变量True和False等价。 (a < b) 返回 true。
>= 大于等于 - 返回x是否大于等于y。 (a >= b) 返回 False。
<= 小于等于 - 返回x是否小于等于y。 (a <= b) 返回 true。

5-8 不只是数字才能做比较运算

不仅number可以做比较运算，list、str、tuple、dict等都是可以做比较的。

  • 比较 str

    • 单个字符串进行比较的时候，是将字符串转化为 ASCII 码之后在进行比较的

      字符串转换成 ASCII 码使用 ord()函数

    • 多个字符串进行比较的时候，是将两组字符串按照索引位顺序取出单个字符进行索引比较的，直至得出比较结果

  • 比较 list、tuple、dict与比较 str是一样的

5-9 逻辑运算符

运算符 逻辑表达式 描述 实例
and x and y 布尔”与” - 如果 x 为 False，x and y 返回 False，否则它返回 y 的计算值。 (a and b) 返回 20。
or x or y 布尔”或” - 如果 x 是非 0，它返回 x 的值，否则它返回 y 的计算值。 (a or b) 返回 10。
not not x 布尔”非” - 如果 x 为 True，返回 False 。如果 x 为 False，它返回 True。 not(a and b) 返回 False

5-10 成员运算符

特点：

  • 判断一个元素是否在一组元素里
  • 返回值是 bool
运算符 描述 实例
in 如果在指定的序列中找到值返回 True，否则返回 False。 x 在 y 序列中 , 如果 x 在 y 序列中返回 True。
not in 如果在指定的序列中没有找到值返回 True，否则返回 False。 x 不在 y 序列中 , 如果 x 不在 y 序列中返回 True。

number、str、list、tuple、set、dict都支持成员运算符

但是 dict的成员运算符比较特别，dict里的成员运算符是针对 key:value里的 key进行操作的。

5-11 身份运算符

运算符 描述 实例
is is 是判断两个标识符是不是引用自一个对象 x is y, 类似 id(x) == id(y) , 如果引用的是同一个对象则返回 True，否则返回 False
is not is not 是判断两个标识符是不是引用自不同对象 x is not y ， 类似 **id(a) != id(b)**。如果引用的不是同一个对象则返回结果 True，否则返回 False。

两个变量的取值相同并不代表身份也相同，身份可以用内存地址表示，因为它们的内存地址不同，所以它们的身份不同。

5-12 如何判断变量的值、身份与类型

在 Python里，一切皆对象。对象的三个特征：

  • 值
  • 身份
  • 类型

Python里有一个专门的函数来判断变量的类型：isinstance()

type()与isinstance()的区别：

type不能判断对象的子类是否属于某种类型的，但是 isinstance可以
