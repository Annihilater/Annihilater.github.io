14-5 None

None在 Python 里表示空。空就是空。空本身也是一个类型，也是一个对象。

  • None ≠ 空字符串
  • None ≠ 空列表
  • None ≠ 0
  • None ≠ False

这个≠有两层意义：

  1. 类型不同
  2. 值不同

空集合：表示集合里没有元素

空元组：表示元组里没有元素

空序列：表示序列里没有元素

None 与 False

  • None 表示不存在的概念，属于 Nonetype 类型
  • False 表示真假，属于 bool 类型

14-7 __len__与__bool__内置方法

__len__方法

  1. 当没有__bool__方法，只有__len__方法的时候，对象的返回结果由__len__的返回结果决定：

    • __len__方法返回 0、False 时，对象的布尔值为 False
    • __len__返回非0整型数字、True 时，对象的布尔值为 True
    • __len__返回非数字、非 bool 类型时，会报错

  2. 我们在外部使用 bool 方法判断对象的时候，实际上是调用了对象内置的__len__方法。

  3. 除了 bool 方法之外还有一个方法可以触发对象内部的__len__方法的调用：len()

    如果一个对象没有内置的__len__方法，强制的求对象的长度是会报错的。

__bool__方法

当__bool__放在存在时，__len__方法的返回结果不再影响对象的布尔值。

  • __bool__方法只能返回布尔类型
