JupyterLab 导出 PDF 中文不显示问题

本文解决方案是基于 macOS 平台，Windows 平台的解决方案大致步骤类似，请自行探索。

今天使用了一下 jupyterlab 导出到 pdf 的功能，发现导出后的 pdf 中文不显示。ipynb 文件如下图所示

导出后的 pdf 为

中文全没了。网上查了下发现，jupyter 暂时不支持中文 pdf 导出。但是事在人为，导出 pdf 显示中文的方法还是有的，解决方案为

  • 先将 ipynb 导出为 latex
  • 在 latex 中添加中文渲染相关的包和字体
  • 再将 latex 渲染为 pdf

ipynb 导出为 latex

解压后可以看到生成的 tex 文件

在 latex 中添加中文渲染相关的包和字体

使用 sublime 打开生成的 matplotlib_example.tex 文件是这样的

需要在 \documentclass{article} 后面插入

其中 \setmainfont{STHeitiSC-Medium} 中的 STHeitiSC-Medium 是 mac 电脑上支持中文的字体。其实这个字体名很好找，打开macOS 上的 字体册 app 即可

如上图选择你中意的中文字体，然后点击左侧上方的小 i 图标就可以在右侧看到中文字体的 PostScript 名称，这个 PostScript 名称就是需要填写在 \setmainfont{PostScript 名称} 内的名称。

将 latex 渲染为 pdf

latex 渲染为 pdf 需要 mactex 的语法支持，因为需要使用 mactex 的 xelatex 命令将 tex 文件转为 pdf。需要先按照 mactex，macOS 上使用 Homebrew 安装 mactex 是非常简单的一条命令就搞定了，Homebrew 真是神器啊。

安装好之后会看到 TeXShop 和 LaTeXiT 已经被安装好了，mactex 安装完成。

接下来使用 mactex 的 xelatex 命令将 tex 文件转为 pdf

可以看到 matplotlib_example.pdf 文件已经生成

再打开 matplotlib_example.pdf 文件可以看到中文已经全部显示出来了，大功告成。
